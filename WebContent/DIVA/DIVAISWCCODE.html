DIVA系統撈出PERFORM LANGUAGE語言別的ISWC CODE

目的:USER要看協會官網內的歌曲庫特定語言別的歌曲資料及ISWC CODE。

前置作業:必須先等DIVA每個月的WEBUPLOAD，依照”把歌曲放上官網的步驟1.(手動)資料預備 2.(程式)刪除歌林曲目 3.(程式)產生IS GROUP 註記 4.(程式)合併欄位 5.(程式) 欄位ot_title內容修改 6.(程式)表格內容修改 7.(程式)刪除三條件曲目/EMI要拉掉曲目 8.(手動)產生的ACCESS檔案匯入SQL SERVER”把當月的歌曲放到官網的DB。確認是最新資料後再開始依照下列步驟操作。

 

步驟:

1.       使用CASH提供的SQL QUERY出歌曲的ISWC CODE:

 select w.worknum, w.worknum_society, w.status, w.perform_language, decode(w.status, 'AV', w.completion, r.distributable) distributable,

       t.e_title, t.c_title, t.genre_detail, i.iswc

from   wrk_work w,

       wrk_work_right r,

       wrk_title t,

       wrk_iswc i

where  w.perform_language in ('ZHO', 'CER', 'HAK', 'YUH')  --這裡是USER提供他們要的語言別

and    w.worknum = r.worknum(+)

and    w.worknum_society = r.worknum_society(+)

and    nvl(r.right_type, 'PER') = 'PER'

and    w.worknum = t.worknum

and    w.worknum_society = t.worknum_society

and    t.sub_title_id = 0

and    w.worknum = i.worknum(+)

and    w.worknum_society = i.worknum_society(+)

and    i.preferred = ‘Y’;

撈出資料後存成CSV檔。

2.  CSV另存成UTF-8編碼，用EXCEL匯入後另存成EXCEL，再把這份EXCEL匯入72那台DB，TABLE_NAME可以自己取(EX:ISWC)。

3.  用72DB裡面的[dbo].[WORKS_IP_STATUS] join用剛匯入的ISWC TABLE:

SELECT distinct a.*,b.iswc FROM WORKS_IP_STATUS a

LEFT JOIN (select distinct worknum,worknum_society,iswc from iswc) b ON  a.WORKNUM=b.worknum and a.WORKNUM_SOCIETY=b.worknum_society

where a.PERFORM_LANGUAGE in ('ZHO','CFR','HAK','YUH')  order by PERFORM_LANGUAGE,worknum,IP_BASE_NO，把查詢結果匯出。
